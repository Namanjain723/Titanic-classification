## 📌 Overview

A classic end-to-end **classification** project: explore the Titanic passenger data, uncover
what drove survival, and predict whether a passenger survived from features like **class,
sex, age and fare**.

## 🗂️ Data
`Titanic-Dataset.csv` — passenger records (Pclass, Sex, Age, SibSp, Parch, Fare, Embarked, Survived).

## 🔬 Approach
1. **Clean** — handle missing values (e.g. age, embarked) and encode categorical features.
2. **EDA** — visualise survival rates by **sex, passenger class and age** with Matplotlib / Seaborn.
3. **Model** — train a classification model to predict the `Survived` target.
4. **Evaluate** — measure accuracy and review which features mattered most.

## 💡 Key takeaways
Survival was far from random — **sex and passenger class** were among the strongest signals, a textbook example of turning raw data into an interpretable predictive model.
